# MVP: фото → 4 персонажа → раскадровка (kie.ai Nano Banana)

> Цель: за одну сессию рабочий локальный MVP двух этапов пайплайна.
> Этап 1: загрузка фото ребёнка → 4 стилизованных персонажа (Pixar, Minecraft, Аниме, LEGO), максимальная узнаваемость лица.
> Этап 2: выбор персонажа → лист раскадровки/character sheet (персонаж с разных ракурсов на одной картинке).
> Плюс: вывод стоимости генераций (учёт COGS).
> API: kie.ai (Nano Banana / Nano Banana Edit), ключ в `.env`.
> Связь со стратегией: это срез шагов 1.4–1.5 из [тех-плана Фазы 1](2026-06-10-faza-1-tech-plan-raskadrovka.md), но через kie.ai вместо Gemini direct (обход биллинг-блокера РФ).

## Фазы

- [x] Ф0. Разведка: спеки kie.ai API (createTask / recordInfo / file upload / модели / цены) — фоновый агент
- [x] Ф1. Каркас: zero-dependency Node-сервер (`mvp/server.js`) + статичный фронт (`mvp/public/index.html`), конфиг цен/моделей/промптов в `mvp/config.js`
- [x] Ф2. Бэкенд: /api/upload (фото → публичный URL через kie.ai upload), /api/characters (4 задачи генерации), /api/task/:id (поллинг), /api/storyboard, /api/usage (стоимость)
- [x] Ф3. Фронтенд: шаг 1 загрузка → грид 4 персонажей с поллингом → выбор → шаг 2 раскадровка; счётчик стоимости
- [x] Ф4. Промпт-инжиниринг узнаваемости: identity-preservation промпты на стиль (лицо, причёска, цвет глаз, родинки, выражение)
- [x] Ф5. E2E-тест на реальном фото (`photo_1944313.jpg`), реальные генерации, проверка стоимости
- [x] Ф6. Ретроспектива + актуализация плана

## Итог

**Реализовано целиком.** Рабочий MVP в `mvp/`: загрузка фото → 4 персонажа (Pixar / Minecraft / Аниме / LEGO, модель google/nano-banana-edit, $0.02/картинка) → выбор → character sheet раскадровка (та же модель, 2 референса: персонаж + оригинал фото). Счётчик стоимости на фронте и `GET /api/usage`. E2E проверен на реальном фото (`photo_1944313.jpg`) через браузер: 4/4 персонажа + раскадровка сгенерированы, узнаваемость подтверждена визуально. Полный проход юзера = $0.10 (5 × $0.02); на все тесты сессии потрачено ~$0.14. Примеры — `mvp/samples/`. Запуск: `PORT=3777 node mvp/server.js` (3000/3001 заняты другими проектами). Ретро: `retrospectives/2026-06-10_18-30_mvp-personazh-raskadrovka.md`.
